# Check for lengthy variable names.

Check for lengthy variable names of variables in a `GADSdat`.

## Usage

``` r
check_var_names_length(GADSdat, boundary = 30)
```

## Arguments

- GADSdat:

  `GADSdat` object.

- boundary:

  Numeric vector of lenght 1. Boundary: Number of characters variables
  names are allowed to have.

## Value

Returns the test report.
