# Check existence of variable labels.

Check the existence of variable labels for all variables in a `GADSdat`.

## Usage

``` r
check_var_labels(GADSdat)
```

## Arguments

- GADSdat:

  `GADSdat` object.

## Value

Returns the test report.

## Details

Checks include

- Have variable labels been assigned?
