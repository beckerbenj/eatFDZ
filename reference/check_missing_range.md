# Check missing range.

Check all labeled values if they fall in the range for missing values
`GADSdat`.

## Usage

``` r
check_missing_range(GADSdat, missingRange = -50:-99)
```

## Arguments

- GADSdat:

  `GADSdat` object.

- missingRange:

  Values which should be declared as missings.
