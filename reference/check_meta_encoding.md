# Check meta data encoding.

Check the occurence of specials signs in the meta data of a `GADSdat`.

## Usage

``` r
check_meta_encoding(GADSdat)
```

## Arguments

- GADSdat:

  `GADSdat` object.

## Value

Returns the test report.

## Details

Checks in the meta data include

- special signs such as `Umlaute`
