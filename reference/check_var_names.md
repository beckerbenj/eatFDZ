# Check variable names.

Check variable names of variables in a `GADSdat`.

## Usage

``` r
check_var_names(GADSdat)
```

## Arguments

- GADSdat:

  `GADSdat` object.

## Value

Returns the test report.

## Details

Checks in variable names include

- special signs such as `Umlaute`

- column names with a `"."` in it.
